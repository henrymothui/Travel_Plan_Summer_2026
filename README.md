# Central Coast Itinerary

Interactive day-by-day trip planner for Hue → Da Nang (Aug 8–12), with checkboxes that save progress in the browser.

## Local development

```bash
npm install
npm run dev
```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000).

## Deploy

Push this repo to GitHub, then import it in [Vercel](https://vercel.com/new). Framework preset: **Next.js**. No env vars required.
